# Cursor Based Image Trail Effect

## Interactive UI Snippets

A collection of modern interactive UI components and creative animations built with React and GSAP.

This project demonstrates:
- Cursor-based image trail animation
- Animated hero product carousel
- Smooth transitions and micro-interactions
- Responsive layout design

![Hero Preview](./git-assets/preview.gif)

## Tech Stack
- React
- GSAP
- Tailwind CSS
- Lucide Icons

## Purpose
To experiment with immersive UI patterns and modern frontend animation techniques.
